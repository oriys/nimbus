Get Started in Minutes

Deploy your first function with just a few commands

1

Start the Platform

Terminal 
# Docker Compose (macOS/Windows)
$ docker compose up -d
$ make run

# Or Kubernetes (OrbStack)
$ cd deployments/k8s/overlays/orbstack
$ ./start.sh
2

Configure CLI

Terminal 
# Initialize CLI configuration
$ nimbus config init --api-url http://localhost:8080

# Verify connection
$ nimbus list
3

Create Your Function

hello.py 
def handler(event):
    name = event.get('name', 'World')
    return {
        'message': f'Hello, {name}!'
    }
4

Deploy & Invoke

Terminal 
# Create function
$ nimbus create hello \
    --runtime python3.11 \
    --handler handler \
    --code hello.py

# Invoke function
$ nimbus invoke hello --data '{"name": "Nimbus"}'
{"message": "Hello, Nimbus!"}

Architecture Overview

Built on proven technologies for reliability and performance

API Gateway

Go-based HTTP gateway with chi router. Handles function management, invocation routing, and authentication.

Scheduler

Distributed task scheduler with NATS messaging. Supports async invocation, cron triggers, and workflow execution.

VM Pool

Firecracker MicroVM pool manager. Pre-warms VMs, handles auto-scaling, and manages VM lifecycle.
