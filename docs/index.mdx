## 基本信息

- 默认服务地址：`http://localhost:8080`（以实际配置的 `server.http_port` 为准）
- API 前缀：`/api/v1`
- 数据格式：JSON（请求/响应）
- 错误格式：`{"error":"..."}`

健康检查：
```bash
curl http://localhost:8080/health
```
